🔔 Discord Webhook

ℹ️ Як отримати Discord Webhook URL:

  1. Відкрий Discord → налаштування каналу (⚙️)
  2. Вкладка Integrations → Webhooks
  3. Натисни New Webhook, дай ім'я
  4. Натисни Copy Webhook URL і встав сюди

📥 Ручний імпорт CSV

Завантаж CSV з Google Форми. Обов'язкові колонки: discord та score. Додатково: nickname, date

📋 Формат CSV (приклад):

discord,nickname,score,date
username#1234,Гравець1,85,2024-01-15
username#5678,Гравець2,92,2024-01-15

🤖 Гайд: Автоматизація через Zapier

Zapier дозволяє автоматично передавати дані з Google Форми в цю таблицю. Коли хтось проходить тест — він одразу з'являється тут!

КРОК 1 — Налаштуй Google Форму

  1. Відкрий свій Google Тест/Форму (або створи новий)
  2. Переконайся що є поле для введення Discord нікнейму (короткий текст)
  3. Активуй автоматичну оцінку в налаштуваннях тесту щоб бали рахувались автоматично
  4. Відкрий Відповіді → натисни іконку Google Таблиць → Створити нову таблицю

КРОК 2 — Зареєструйся в Zapier

  1. Зайди на zapier.com → безкоштовний план (до 100 зап/міс)
  2. Створи акаунт або увійди через Google
  3. Натисни Create Zap (створити автоматизацію)

КРОК 3 — Trigger (Тригер): Google Sheets

  1. В полі "Trigger App" знайди та вибери Google Sheets
  2. В "Event" вибери New Spreadsheet Row (нова строка в таблиці)
  3. Підключи свій Google акаунт
  4. Вибери таблицю яку ти створив для форми і відповідний аркуш
  5. Натисни Test Trigger — має показати тестові дані з форми

КРОК 4 — Action (Дія): Webhooks by Zapier

  1. Додай дію → знайди Webhooks by Zapier
  2. Вибери POST
  3. В полі URL встав: https://твій-сайт/agitation-webhook
    ⚠️ Оскільки сайт чисто HTML — використовуй Make.com webhooks або пряме з'єднання (дивись нижче)
  4. В Data (тип: Form) додай поля:
    discord → вибери колонку Discord з Google Sheets
    nickname → колонку з нікнеймом
    score → колонку з балами (Оцінка/Score)
  5. Натисни Test Action

💡 Альтернатива: Make.com (безкоштовніше)

  1. Зайди на make.com → безкоштовний план (1000 операцій/міс)
  2. Створи новий сценарій: Google Sheets → Webhooks
  3. Make дає тобі webhook URL — скопіюй його
  4. Вкажи його в налаштуваннях цієї сторінки (⚙️ Налаштування → Zapier Webhook URL)
  5. Тепер кожен новий рядок в Google Sheets автоматично надсилається сюди

🔵 Крок 5 — Сповіщення в Discord

  1. В Zapier/Make додай ще одну дію: Discord → Send Channel Message via Webhook
  2. Або використай Webhooks by Zapier → POST на твій Discord Webhook URL
  3. Вибери формат повідомлення — можна з ім'ям кандидата та балами
  4. Готово! Бот буде кидати повідомлення автоматично
