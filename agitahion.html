📖 Гайд: Zapier + Google Forms → Firebase
1

Створіть аккаунт на Zapier

Перейдіть на zapier.com → зареєструйтесь безкоштовно. Безкоштовний план підтримує до 100 запусків/місяць, що достатньо для тестів.

2

Натисніть "Create Zap"

Зліва у меню → Create → Zap. Ви побачите редактор де є Trigger (подія що запускає) і Action (дія).

3

Trigger: Google Forms

Виберіть додаток Google Forms → подія "New Form Response".
Підключіть свій Google-акаунт → виберіть потрібну форму (де є Discord нік і бали).
Натисніть "Test trigger" — Zapier підтягне зразок відповіді.

4

Action: Firebase / Webhooks

Варіант A (Простий): Виберіть Webhooks by Zapier → POST.
URL: https://firestore.googleapis.com/v1/projects/ВАШ_PROJECT_ID/databases/(default)/documents/examResults
Headers: Content-Type: application/json

Body (JSON): 

{
  "fields": {
    "discordNick": {"stringValue": "{{відповідь про Discord нік}}"},
    "discordId":   {"stringValue": "{{відповідь про Discord ID}}"},
    "score":       {"integerValue": "{{відповідь про бали}}"},
    "testName":    {"stringValue": "Назва вашого тесту"},
    "date":        {"stringValue": "{{timestamp}}"},
    "agitated":    {"booleanValue": false},
    "source":      {"stringValue": "google_forms"}
  }
}
5

Firebase Auth для Zapier (щоб можна було записувати)

У Firebase Console → Authentication → Sign-in methods → увімкніть Anonymous.
Або у Firestore Rules тимчасово (тільки для налаштування!) поставте: 

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /examResults/{doc} {
      allow write: if true; // ТІЛЬКИ ДЛЯ ТЕСТУ!
      allow read: if request.auth != null;
    }
  }
}

⚠️ Після налаштування поверніть нормальні rules!

6

Назви полів у Google Формі

Щоб Zapier правильно розпізнав поля, у формі назвіть питання так:
• Discord нік або Ваш Discord Username
• Discord ID (або можна без нього — тоді не заповнюється)
• Результат / Бали — якщо форма не рахує автоматично, можна вручну вписувати або використовувати Google Apps Script.

7

Увімкніть Zap і тестуйте

Натисніть Publish → Zap активний. Заповніть тестову форму — через 1–2 хвилини запис з'явиться у таблиці на цій сторінці.
Після цього натисніть "📢 Кинути агіт" — повідомлення полетить у Discord-канал!
